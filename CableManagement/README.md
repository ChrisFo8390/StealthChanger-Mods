# Umbilical plate with CNLinko and PTFE fitting

Modded [N3MI-DG's umbilical system](https://github.com/DraftShift/CableManagement/tree/main/UserMods/N3MI-DG/Umbilical_plates_V2) so I can simply remove a toolhead without demounting anything in the back.

For now only as 5 tool variant.</br>
ATTENTION: The size and the angle of the push fitting prevent this plate from being mounted directly on back top extrusion and the backpanel needs a larger cutout!!!
